Jakarta servlet

Em Java, um servlet é uma classe que estende as capacidades de um servidor. Os servlets podem responder a diversos tipos de solicitações, mas geralmente são utilizados para estender as aplicações hospedadas por servidores web. Para essas aplicações, a tecnologia Jakarta Servlet define classes servlet específicas para HTTP.

Os pacotes javax.servlet e javax.servlet.http fornecem interfaces e classes para escrever servlets. Todos os servlets devem implementar a interface Servlet, que define os métodos de ciclo de vida.
